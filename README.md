# Netflix Clone

Clone Netflix using Next.js


## Demo

https://netflix-clone-linhdangquang.vercel.app/


## Features

- Login/Sign up
- Stripe Subscription, cancel, change
- Youtube Trailer
- Add to list


## Tech Stack

**Client:** React, Next.js, Recoil, TailwindCSS, MUI

**Server:** Firebase, Stripe payment, tmdb api
